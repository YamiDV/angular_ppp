📝 Cómo elaborar tu CV para una práctica en TI

El CV es tu carta de presentación. Aquí te damos una guía clara y práctica para construir un CV efectivo que te ayude a conseguir tu primer trabajo como practicante.

🎯 1. Datos personales claros

  • Nombre completo
  • Correo profesional (evita apodos o informales)
  • Teléfono
  • LinkedIn y/o GitHub (si tienes)

💡 2. Perfil profesional

Estudiante de Ingeniería de Sistemas apasionado por la ciberseguridad y el desarrollo web...

📚 3. Educación

  • Nombre de tu universidad
  • Carrera y ciclo actual

🛠️ 4. Habilidades técnicas

  • Python, HTML/CSS/JS, SQL, Git, Power BI
  • Linux básico, metodologías ágiles

📁 5. Proyectos o experiencia

  • Proyectos universitarios destacados
  • Contribuciones a GitHub

✨ 6. Soft Skills

  • Trabajo en equipo
  • Comunicación
  • Responsabilidad

✅ 7. Consejos adicionales

  • No más de 1 hoja
  • Formato PDF
  • Diseño limpio: Canva, Novoresume, Word
